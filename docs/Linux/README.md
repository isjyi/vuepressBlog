# 文件/文件夹管理 

`ls` 

>列出当前目录文件（不包括隐含文件)

`ls -a` 

>列出当前目录文件（包括隐含文件） 

`ls -l` 

>列出当前目录下文件的详细信息 

`cd ..` 

>回当前目录的上一级目录 

`cd -` 

>回上一次所在的目录 

`cd ~ 或 cd` 

>回当前用户的宿主目录 

`mkdir 目录名`
 
>创建一个目录
 
`rmdir 空目录名`
 
>删除一个空目录 

`rm` 

>文件名 文件名 删除一个文件或多个文件 

`rm -rf`
 
>非空目录名 删除一个非空目录下的一切 

`mv 路经/文件 /路径/文件`

>移动相对路经下的文件到绝对路经下 

`mv 文件名 新名称`
 
>在当前目录下改名 

`find 路经 -name “字符串”`
 
>查找路经所在范围内满足字符串匹配的文件和目录
